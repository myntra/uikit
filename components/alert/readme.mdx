## Appearances

Alerts are used to convey messages to the user. There are different types of alerts:

```jsx preview
<>
  <Alert type="error">An example of error alert.</Alert>
  <Alert type="warning">An example of warning alert.</Alert>
  <Alert type="success">An example of success alert.</Alert>
  <Alert type="primary">An example of primary alert.</Alert>
</>
```

## Dismissable Alert

```jsx preview
const [isVisible, setVisible] = useState(true);

return isVisible &&
  <Alert type="warning" onClose={() => setVisible(false)}>
    <div>{'CAUTION! The image you are trying to upload has invalid format. It might not render well.'}</div>
    <Button type="link" inheritTextColor>Proceed Anyway</Button>
  </Alert>
```
